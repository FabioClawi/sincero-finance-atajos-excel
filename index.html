Recursos para Excel

Sincero Finance

Funciones (F1–F12)

  • F1 – Muestra la ayuda de Excel
  • F2 – Entra en modo de edición para la celda activa
  • F3 – Muestra el cuadro de diálogo «Pegar nombre» si existe un nombre definido
  • F4 – Repite la última acción
  • F5 – Abre el cuadro de diálogo «Ir a»
  • F6 – Mueve el foco entre los paneles de un libro dividido
  • F7 – Inicia la revisión ortográfica
  • F8 – Activa el modo Ampliar selección
  • F9 – Recalcula las fórmulas en los libros abiertos
  • F10 – Activa la barra de menús
  • F11 – Crea una hoja de gráfico con el rango de celdas seleccionado
  • F12 – Abre el cuadro de diálogo «Guardar como»

Atajos de una sola tecla

  • AvPág – Desplazarse una pantalla hacia abajo dentro de la hoja
  • RePág – Desplazarse una pantalla hacia arriba dentro de la hoja
  • Alt – Activa la barra de menús
  • Enter – Acepta la entrada de una celda y selecciona la celda inferior
  • Esc – Cancela cuadros de diálogo y la entrada de una celda
  • Barra espaciadora – Activa o desactiva las casillas de verificación en cuadros de diálogo
  • Fin – Activa o desactiva el «Modo final»
  • Inicio – Moverse al principio de la fila de la celda activa
  • Retroceso – Borra el contenido de una celda y entra en el modo de edición
  • Suprimir – Borra el contenido de una celda
  • Tabulador – Completa la entrada de una celda y selecciona la celda a la derecha
  • Flechas – Moverse a las celdas de arriba, abajo, izquierda o derecha

Ctrl + tecla de función

  • Ctrl + F1 – Oculta o muestra la cinta de opciones
  • Ctrl + F2 – Muestra el cuadro de diálogo Imprimir
  • Ctrl + F3 – Muestra el Administrador de nombres
  • Ctrl + F4 – Cierra la ventana del libro actual
  • Ctrl + F5 – Restaura el tamaño de la ventana del libro actual
  • Ctrl + F6 – Moverse al siguiente libro abierto
  • Ctrl + F7 – Permite mover la ventana del libro cuando la ventana no está maximizada
  • Ctrl + F8 – Permite cambiar el tamaño de la ventana del libro cuando la ventana no está maximizada
  • Ctrl + F9 – Minimiza la ventana del libro actual
  • Ctrl + F10 – Maximiza la ventana de un libro minimizado
  • Ctrl + F11 – Crea una hoja de macros
  • Ctrl + F12 – Muestra el cuadro de diálogo Abrir

Alt + tecla de función

  • Alt + F1 – Crea un gráfico incrustado con los datos del rango actual
  • Alt + F2 – Abre el cuadro de diálogo «Guardar como»
  • Alt + F4 – Cierra Excel
  • Alt + F8 – Muestra el cuadro de diálogo «Macro»
  • Alt + F10 – Abre el panel «Selección y visibilidad»
  • Alt + F11 – Abre el editor de Visual Basic para Aplicaciones

Shift (Mayús) + tecla de función

  • Mayús + F2 – Añade un comentario en la celda
  • Mayús + F3 – Abre el cuadro de diálogo «Insertar función» (al editar fórmulas muestra «Argumentos de función»)
  • Mayús + F4 – Repite la operación «Buscar siguiente» en una búsqueda
  • Mayús + F5 – Abre el cuadro de diálogo «Buscar»
  • Mayús + F6 – Cambia entre la hoja, la cinta de opciones, la barra de estado y el panel de tareas
  • Mayús + F7 – Abre el panel de tareas «Referencia»
  • Mayús + F8 – Agrega un rango de celdas a la selección con las teclas de dirección
  • Mayús + F9 – Calcula todas las fórmulas de la hoja activa
  • Mayús + F10 – Muestra el menú contextual de un elemento seleccionado
  • Mayús + F11 – Inserta una nueva hoja de cálculo
  • Mayús + F12 – Abre el cuadro de diálogo «Guardar cómo»

Ctrl + Shift + …

  • Ctrl + Mayús + F3 – Abre el cuadro de diálogo «Crear nombres a partir de selección»
  • Ctrl + Mayús + F6 – Cambia al libro anterior abierto
  • Ctrl + Mayús + F10 – Activa la barra de menús
  • Ctrl + Mayús + F12 – Abre el cuadro de diálogo «Imprimir»
  • Ctrl + Mayús + F – Abre el menú «Formato de celdas» en la pestaña Fuentes
  • Ctrl + Mayús + L – Activa o desactiva los filtros en un rango
  • Ctrl + Mayús + O – Selecciona todas las celdas con comentarios
  • Ctrl + Mayús + U – Amplía la barra de fórmulas
  • Ctrl + Mayús + Entrar – Ingresa una fórmula como fórmula matricial
  • Ctrl + Mayús + Inicio – Extiende la selección hasta el comienzo de la hoja
  • Ctrl + Mayús + Fin – Extiende la selección hasta la última celda utilizada
  • Ctrl + Mayús + AvPág – Selecciona la hoja actual y la siguiente de un libro
  • Ctrl + Mayús + RePág – Selecciona la hoja actual y la anterior de un libro
  • Ctrl + Mayús + Dirección – Extiende la selección a la última celda no vacía de la columna o fila actual
  • Ctrl + Mayús + Espacio – Selecciona toda la hoja de cálculo

Atajos para fórmulas y datos

  • Alt + = – Inserta autosuma
  • Alt + Abajo – Abre una lista de validación de datos
  • Alt + Enter – Inserta un salto de línea dentro de una celda
  • Ctrl + , – Inserta la fecha actual
  • Ctrl + : – Inserta la hora actual
  • Ctrl + " – Copia el valor de la celda de arriba
  • Ctrl + ’ – Copia la fórmula de la celda de arriba
  • Ctrl + Enter – Rellena el rango seleccionado con la entrada actual
  • Ctrl + Suprimir – Borra el texto hasta el final de la línea
  • Ctrl + Alt + K – Inserta un hipervínculo
  • Mayús + Enter – Completa la entrada de la celda y va a la celda superior
  • Enter – Completa la entrada de la celda y va a la celda inferior
  • Mayús + Tabulador – Completa la entrada de la celda y va a la celda de la izquierda
  • Tabulador – Completa la entrada de la celda y va a la celda de la derecha

Atajos para seleccionar datos

  • Mayús + Espacio – Selecciona la fila actual
  • Ctrl + Espacio – Selecciona la columna actual
  • Mayús + Inicio – Extiende la selección hasta el inicio de la fila
  • Mayús + AvPág – Extiende la selección hacia abajo
  • Mayús + RePág – Extiende la selección hacia arriba
  • Mayús + Dirección – Extiende la selección una celda hacia arriba, abajo, izquierda o derecha
  • Ctrl + Mayús + Dirección – Extiende la selección a la última celda no vacía de la columna o fila actual
  • Ctrl + Mayús + Inicio – Extiende la selección hasta la primera celda usada de la hoja
  • Ctrl + Mayús + Fin – Extiende la selección hasta la última celda utilizada de la hoja
  • Ctrl + Mayús + * – Selecciona el área actual alrededor de la celda activa

Atajos de teclado comunes

  • Ctrl + C – Copiar el contenido de las celdas seleccionadas
  • Ctrl + V – Pegar el contenido del portapapeles
  • Ctrl + X – Cortar las celdas seleccionadas
  • Ctrl + Z – Deshacer la última acción
  • Ctrl + Y – Rehacer la última acción deshecha
  • Ctrl + S – Guardar el libro actual
  • Ctrl + F – Buscar y reemplazar
  • Ctrl + Flecha – Moverse rápidamente entre celdas en la dirección de la flecha

Lógicas y manejo de errores
  • SI — Evalúa una condición y devuelve un valor si es verdadera y otro si es falsa. Patrón: SI(condición, valor_si_verdadero, valor_si_falso)
  • Y — Devuelve VERDADERO si todas las condiciones son verdaderas. Patrón: Y(cond1, cond2,…)
  • O — Devuelve VERDADERO si alguna condición es verdadera. Patrón: O(cond1, cond2,…)
  • SI.ERROR — Devuelve un valor alternativo cuando una expresión genera un error. Patrón: SI.ERROR(expr, valor_si_error)
Texto
  • TEXTO — Da formato a un número como texto. Patrón: TEXTO(valor, "formato")
  • CONCAT / CONCATENAR — Une textos de varias celdas en una sola. Patrón: CONCAT(texto1, texto2,…)
  • IZQUIERDA — Extrae N caracteres desde la izquierda de un texto【703935696952100†L148-L152】. Patrón: IZQUIERDA(texto, n)
  • DERECHA — Extrae N caracteres desde la derecha de un texto【584597639123524†L145-L151】. Patrón: DERECHA(texto, n)
  • LARGO — Devuelve la longitud de una cadena de texto【705381333412676†L135-L146】. Patrón: LARGO(texto)
  • HALLAR — Devuelve la posición inicial de un texto dentro de otro texto【718109329976163†L145-L168】. Patrón: HALLAR(texto_buscado, dentro_del_texto, [inicio])
  • SUSTITUIR — Sustituye texto específico por texto nuevo en una cadena【481778781079754†L140-L143】. Patrón: SUSTITUIR(texto, texto_original, texto_nuevo, [número_ocurrencia])
  • EXTRAE — Devuelve un número específico de caracteres de una cadena de texto, comenzando en la posición indicada【935525875290949†L138-L141】. Patrón: EXTRAE(texto, inicio, núm_caracteres)
  • MAYUSC — Convierte todas las letras de un texto a mayúsculas. Patrón: MAYUSC(texto)
  • MINUSC — Convierte todas las letras de un texto a minúsculas. Patrón: MINUSC(texto)
  • NOMPROPIO — Convierte la primera letra de cada palabra a mayúscula y las demás a minúscula. Patrón: NOMPROPIO(texto)
  • REEMPLAZAR — Reemplaza parte de un texto por otro a partir de una posición y longitud. Patrón: REEMPLAZAR(texto, inicio, num_caracteres, texto_nuevo)
  • ESPACIOS — Elimina los espacios innecesarios de un texto, dejando solo un espacio entre palabras. Patrón: ESPACIOS(texto)
Búsqueda y referencia
  • BUSCARV — Busca un valor en la primera columna de una tabla y devuelve un valor de la misma fila. Patrón: BUSCARV(valor, tabla, col, [aprox])
  • BUSCARX — Realiza una búsqueda flexible en filas o columnas. Patrón: BUSCARX(valor, buscar_en, devolver_desde, [si_no], [modo])
  • INDICE — Devuelve el valor de una celda en un rango según la fila y la columna especificadas. Patrón: INDICE(rango, fila, [col])
  • COINCIDIR — Devuelve la posición de un valor dentro de un rango. Patrón: COINCIDIR(valor, rango, [tipo])
  • SUMAPRODUCTO — Calcula la suma de los productos de las entradas correspondientes en dos o más matrices【418171428656056†L135-L137】. Patrón: SUMAPRODUCTO(matriz1, matriz2,…)
  • INDIRECTO — Devuelve la referencia especificada por una cadena de texto. Patrón: INDIRECTO(ref_texto, [a1])
  • DESREF — Devuelve una referencia desplazada desde una celda inicial; puede devolver una celda o un rango. Patrón: DESREF(ref, filas, columnas, [alto], [ancho])
  • VALOR.NUMERO — Convierte una cadena de texto que representa un número en un valor numérico independientemente de la configuración regional【153604347766644†L138-L142】. Patrón: VALOR.NUMERO(texto, [sep_decimal], [sep_grupos])
Matemáticas y estadística
  • SUMA — Suma valores individuales, referencias o rangos de celdas. Patrón: SUMA(rango)
  • PROMEDIO — Devuelve la media aritmética de un rango de números. Patrón: PROMEDIO(rango)
  • MAX — Devuelve el valor máximo de un conjunto de valores【325043969870284†L150-L154】. Patrón: MAX(número1, número2,…)
  • MIN — Devuelve el valor mínimo de un conjunto de valores. Patrón: MIN(número1, número2,…)
  • REDONDEAR — Redondea un número a la cantidad de decimales especificada. Patrón: REDONDEAR(número, decimales)
  • CONTAR — Cuenta las celdas que contienen números. Patrón: CONTAR(rango)
  • CONTARA — Cuenta celdas no vacías en un rango【664325217068802†L25-L55】. Patrón: CONTARA(rango)
  • ABS — Devuelve el valor absoluto de un número, es decir, el número sin su signo【27004298100864†L139-L141】. Patrón: ABS(número)
  • PRODUCTO — Devuelve el producto de todos los números proporcionados. Patrón: PRODUCTO(número1, número2,…)
  • MOD — Devuelve el resto después de dividir un número entre el divisor. Patrón: MOD(número, divisor)
  • POTENCIA — Devuelve el resultado de elevar un número a una potencia. Patrón: POTENCIA(base, exponente)
  • RAIZ — Devuelve la raíz cuadrada positiva de un número. Patrón: RAIZ(número)
  • INT — Devuelve la parte entera de un número decimal redondeando hacia abajo. Patrón: INT(número)
  • JERARQUIA — Devuelve la posición (ranking) de un número dentro de una lista de números【506982389865431†L150-L152】. Patrón: JERARQUIA(número, rango, [orden])
  • ALEATORIO.ENTRE — Devuelve un número entero aleatorio entre los valores inferior y superior especificados. Patrón: ALEATORIO.ENTRE(inferior, superior)
  • REDONDEAR.MAS — Redondea un número hacia arriba, alejándolo de cero【315128192888800†L140-L142】. Patrón: REDONDEAR.MAS(número, núm_decimales)
  • REDONDEAR.MENOS — Redondea un número hacia abajo, acercándolo a cero【26297301274386†L138-L142】. Patrón: REDONDEAR.MENOS(número, núm_decimales)
  • MEDIANA — Devuelve la mediana de los números dados; es el valor que se encuentra en medio de un conjunto de números【128893424147305†L148-L151】. Patrón: MEDIANA(número1, número2, …)
  • MODA — Devuelve el valor que se repite con mayor frecuencia en un rango de datos【428865673112129†L135-L143】. Patrón: MODA(número1, número2, …)
  • DESVEST — Calcula la desviación estándar de una muestra【710882887800549†L135-L136】. Patrón: DESVEST(número1, número2, …)
  • ALEATORIO — Devuelve un número real aleatorio mayor o igual que 0 y menor que 1【228581187812171†L148-L152】. Patrón: ALEATORIO()
  • SUBTOTALES — Devuelve un subtotal en una lista o base de datos【4819453831502†L148-L154】. Patrón: SUBTOTALES(núm_función, ref1, [ref2], …)
Condicionales
  • SUMAR.SI — Suma los valores de un rango que cumplen una condición【426146774332217†L145-L149】. Patrón: SUMAR.SI(rango, criterio, [rango_suma])
  • SUMAR.SI.CONJUNTO — Suma los valores de un rango que cumplen varios criterios【273862720520747†L145-L149】. Patrón: SUMAR.SI.CONJUNTO(rango_suma, rango_crit1, crit1, …)
  • CONTAR.SI — Cuenta las celdas en un rango que cumplen una condición【728311611195737†L135-L167】. Patrón: CONTAR.SI(rango, criterio)
  • CONTAR.SI.CONJUNTO — Cuenta las celdas que cumplen varios criterios【15702600221654†L145-L166】. Patrón: CONTAR.SI.CONJUNTO(rango1, criterio1, rango2, criterio2,…)
  • PROMEDIO.SI — Calcula el promedio de los valores en un rango que cumplen una condición【685176714977804†L148-L151】. Patrón: PROMEDIO.SI(rango_promedio, rango_criterio, criterio)
  • CONTAR.BLANCO — Cuenta el número de celdas vacías en un rango. Patrón: CONTAR.BLANCO(rango)
  • SI.CONJUNTO — Comprueba múltiples condiciones y devuelve el valor correspondiente a la primera condición verdadera【950934008105965†L145-L148】. Patrón: SI.CONJUNTO(cond1, valor1, cond2, valor2, …)
  • PROMEDIO.SI.CONJUNTO — Devuelve el promedio de las celdas que cumplen varios criterios【337145431849388†L148-L152】. Patrón: PROMEDIO.SI.CONJUNTO(rango_promedio, rango_crit1, crit1, …)
Fecha y hora
  • HOY — Devuelve la fecha actual. Patrón: HOY()
  • AHORA — Devuelve la fecha y hora actual. Patrón: AHORA()
  • DIAS — Devuelve el número de días entre dos fechas【270472287350548†L139-L142】. Patrón: DIAS(fecha_final, fecha_inicial)
  • DIASEM — Devuelve el día de la semana como un número entre 1 (domingo) y 7 (sábado)【151521196237895†L150-L155】. Patrón: DIASEM(fecha, [tipo])
  • DIA.LAB — Calcula una fecha laboral desplazada, excluyendo fines de semana y festivos. Patrón: DIA.LAB(inicio, días, [festivos])
  • MES — Devuelve el mes de una fecha representada por un número de serie (1 = enero, 12 = diciembre)【353771291556841†L138-L141】. Patrón: MES(núm_de_serie)
  • AÑO — Devuelve el año correspondiente a una fecha como número de cuatro dígitos【850118950635223†L140-L142】. Patrón: AÑO(núm_de_serie)
  • FECHA — Combina año, mes y día en una fecha válida【160847189372569†L145-L155】. Patrón: FECHA(año, mes, día)
  • SIFECHA — Calcula el número de días, meses o años entre dos fechas【833624702460401†L145-L146】. Patrón: SIFECHA(fecha_inicial, fecha_final, unidad)
